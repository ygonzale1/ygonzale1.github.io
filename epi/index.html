📊 How this tool works

🎯 Purpose

Analyze the relationship between air pollution (PM₂.₅, NO₂, O₃, etc.) and epidemiological incidence rates. Explore delayed effects (lag), detect extreme incidence peaks, and visualize seasonal trends.

📁 Data requirements

  • date (YYYY-MM-DD) and incidencia_1000 (incidence per 1000) are mandatory.
  • Optional: pm2.5, pm10, no2, nox, o3, so2.
  • CSV format with header, commas or semicolons supported.

🚀 How to use

  1. Select a preloaded city dataset or upload your own CSV using "Browse".
  2. Dashboard shows key metrics, trends, and correlation analysis.
  3. Adjust time filters to focus on specific periods.
  4. Explore lagged correlations (0–7 days) for delayed impact.
  5. Peaks table shows statistically unusual incidence days.
  6. Monthly bar chart reveals seasonal patterns.

📈 Correlation & p-values

Pearson's r + significance. p < 0.05 indicates significant association. Lag analysis reveals delayed effects.

⚠️ Peak detection

Threshold = mean + 2×standard deviation. Helps identify outbreak episodes.

💡 Tip: For local files use "Browse". Preloaded datasets require a local server (e.g., python -m http.server).
